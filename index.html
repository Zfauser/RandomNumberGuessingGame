Rules

If you select "Easy" (Default Mode) mode you have 5 tries to pick a number between 1 and 10

If you select "Medium" mode you have 3 tries to pick a number between 1 and 15

If you select "Hard" mode you have 3 tries to pick a number between 1 and 20
